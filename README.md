# MiSTer Wallpapers Collection

A curated collection of custom wallpapers for MiSTer FPGA.

All wallpapers in this repository are created by me and designed for MiSTer.

## Features

- Custom-made wallpapers for MiSTer FPGA
- Designed specifically for MiSTer setups
- Compatible with MiSTer Companion wallpaper installer

## Usage

### Manual Install
1. Download your preferred wallpaper
2. Copy it to your MiSTer `/media/fat/wallpapers` directory

### MiSTer Companion - Wallpapers Tab
- Install & Update wallpapers semi-automatically to your MiSTer with MiSTer Companion

### Update_all.
- Select Anime0t4ku Wallpapers in the Extra options in update_all
